# Gerador de Propostas Comerciais

Aplicacao offline para montar propostas profissionais com preview em tempo real, templates visuais e exportacao via imprimir/salvar PDF.

## Funcionalidades
- Formulario completo de cliente, itens e condicoes
- Itens dinamicos com subtotal e total automatico
- Templates Clean, Premium e Neon
- Copiar resumo para compartilhar
- Auto-save no localStorage
- Exportar para PDF via impressao

## Como rodar
- Abra o `index.html` no navegador
- Ou use Live Server no VS Code

## Exportar PDF
- Clique em **Imprimir/PDF** e escolha "Salvar como PDF" no navegador

## Hospedagem gratis
### GitHub Pages
1. Crie um repositorio no GitHub
2. Envie os arquivos do projeto
3. Em **Settings â†’ Pages**, selecione `main` e `/root`
4. Salve e aguarde a URL publicar

### Netlify
1. Acesse o Netlify e faca login
2. Arraste a pasta do projeto para a area de deploy
3. O link publico sai em segundos

## Personalizacao
- Emitente e dados: clique em **Config** e altere os campos
- Logo: substitua `assets/img/logo-placeholder.svg`
- Templates: ajuste cores no `styles.css`
