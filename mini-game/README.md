Mini game mobile-first em Canvas. Um toque inverte o sentido do orbe enquanto voce desvia dos drones.

## Como jogar
- Toque na tela para inverter a direcao.
- Desvie dos drones e colete energia para manter o combo.

## Como rodar
- Abra o `index.html` no navegador

## Personalizacao
- Ajuste cores em `styles.css`.
- Ajuste velocidade em `app.js` (state.speed).
