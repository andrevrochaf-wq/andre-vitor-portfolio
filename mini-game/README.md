# Mini Game — Neon Drift

Mini jogo arcade em Canvas para portfólio. Offline, leve e divertido.

## Como jogar
- Mova o mouse ou toque para controlar o orbe.
- Desvie dos drones e colete orbes para aumentar o combo.
- Pegue o escudo para sobreviver a um impacto.

## Como rodar
- Abra o `index.html` no navegador

## Hospedagem (GitHub Pages)
1. Crie um repositorio
2. Envie a pasta `mini-game`
3. Settings → Pages → main /root

## Personalizacao
- Ajuste cores em `styles.css` (`:root`).
- Ajuste velocidades em `app.js`.
