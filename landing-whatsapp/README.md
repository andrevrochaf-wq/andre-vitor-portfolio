# Landing de Alta Conversao + WhatsApp

Landing page premium com CTA para WhatsApp e gerador de mensagem para acelerar a conversao.

## Como trocar telefone
- Edite a constante `WA_PHONE` em `app.js`.

## Funcionalidades
- CTA principal para WhatsApp
- Gerador de mensagem com copiar e abrir WhatsApp
- FAQ com accordion acessivel
- Scrollspy, reveal e reduzir animacoes
- Botao flutuante do WhatsApp no mobile

## Como rodar
- Abra o `index.html` no navegador
- Ou use Live Server no VS Code

## Hospedagem gratis
### GitHub Pages
1. Crie um repositorio
2. Envie os arquivos
3. Settings → Pages → main /root

### Netlify
1. Faça login
2. Arraste a pasta do projeto

## Personalizacao
- Textos: edite em `index.html`
- Cores: ajuste em `styles.css` (`:root`)
- WhatsApp: altere em `app.js`
